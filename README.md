## 📦 What crystal-server does

crystal-server is a command and control server for CrystalC2. It gives you a place to start, manage, and connect your CrystalC2 setup on Windows.

Use it when you want a single server app to handle your control side. It keeps the process simple and helps you get started without a complex install.

## 🖥️ System requirements

Before you download, check that your PC has:

- Windows 10 or Windows 11
- 64-bit support
- At least 4 GB of RAM
- 200 MB of free disk space
- Internet access for the first download
- Admin rights if Windows asks for them

For best results, use a machine that stays on while the server runs. A laptop can work, but a desktop is a better fit.

## 🛠️ Troubleshooting

### The file will not open

- Make sure you downloaded the Windows file from the releases page
- If the file is in a ZIP, extract it first
- Check that the app file is still inside the folder
- Try right-clicking the file and choosing Run as administrator

### Windows says it blocked the app

- Open the file again and look for a More info button
- Choose Run anyway if you trust the download source
- Check your antivirus if it still stops the app

### The server will not start

- Make sure another app is not using the same port
- Close other server tools that may conflict
- Restart the app
- Restart Windows if needed

### The app starts but does not connect

- Check your network
- Make sure the correct port is open
- Confirm the server address is right
- Try a local test on the same machine first

### The window closes right away

- Open the app from its folder, not from a shortcut that may be broken
- Make sure all files from the release stay in the same folder
- Look for an error message in the logs
